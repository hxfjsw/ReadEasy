# ReadEasy需求文档

## 1. 项目概述

### 1.1 项目背景
ReadEasy是一款专为英语学习者设计的 Windows 桌面应用程序，旨在帮助用户更流畅地阅读外文文本。通过解析电子书（Epub）和网页内容，识别、提取并分析生词，帮助用户建立个性化的单词本，提升阅读理解能力。

### 1.2 目标用户
- 英语学习者（初中级到高级）
- 需要阅读英文原版书籍的用户
- 备考雅思、托福、GRE 等考试的学生
- 希望扩大词汇量的英语爱好者

### 1.3 核心价值
- **降低阅读门槛**：自动标注生词，减少查词打断阅读的情况
- **个性化学习**：基于用户词汇量定制生词识别
- **高效积累词汇**：边阅读边收集生词，形成个人单词本

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 词汇分析模块
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 文本解析 | 解析 MOBI、TXT 格式电子书及网页内容 | P0 |
| 词汇分级 | 基于词频/难度对单词进行分级（如：小学、初中、高中、四级、六级、雅思、托福、GRE） | P0 |
| 生词高亮 | 根据用户设定的词汇水平，自动高亮显示生词 | P0 |
| 词性标注 | 显示单词词性（名词、动词、形容词等） | P1 |

#### 2.1.2 单词提取模块
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 音标显示 | 显示英式/美式音标 | P0 |
| 释义查询 | 显示中英文释义 | P0 |
| 例句展示 | 提供单词在上下文中的例句 | P1 |
| 发音播放 | 点击播放单词发音（TTS 或在线音频） | P1 |

#### 2.1.3 单词本管理模块
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 生词收藏 | 一键收藏生词到个人单词本 | P0 |
| 单词本分类 | 支持创建多个单词本（按书籍、时间、主题分类） | P1 |
| 导出功能 | 支持导出为 Anki、Excel、PDF 格式 | P1 |
| 复习提醒 | 基于艾宾浩斯遗忘曲线提醒复习 | P2 |

#### 2.1.4 阅读器模块
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 电子书阅读 | 支持 MOBI、TXT 格式阅读 | P0 |
| 网页阅读 | 支持粘贴网页 URL 或 HTML 内容解析 | P1 |
| 阅读进度 | 记录阅读进度和书签 | P1 |
| 主题切换 | 支持白天/夜间模式、字体大小调整 | P1 |

### 2.2 用户设置

#### 2.2.1 词汇水平设置
- 预设词汇等级：小学、初中、高中、四级、六级、考研、雅思、托福、GRE、专八
- 自定义词汇量：用户可导入已掌握的单词列表
- 词汇测试：通过简单测试评估用户词汇水平

#### 2.2.2 界面设置
- 字体选择（多种英文字体）
- 字体大小调节
- 行间距调节
- 主题颜色（浅色/深色/护眼模式）
- 高亮颜色自定义

## 4. 技术架构

### 4.1 技术选型
| 层级 | 技术方案 |
|------|---------|
| 开发框架 | Electron + React  |
| UI 框架 | React + Tailwind CSS / Ant Design |
| 状态管理 | Zustand / Redux Toolkit |
| 本地存储 | SQLite  |
| 电子书解析 | epub.js / 自定义 MOBI 解析器 |
| 词典服务 | 本地词典库 + 在线 API 备选 |

### 4.2 数据模型

#### 用户表 (User)
```
- id: 主键
- vocabulary_level: 词汇等级
- custom_words: 自定义已掌握单词
- settings: 用户设置 JSON
- created_at: 创建时间
```

#### 单词表 (Word)
```
- id: 主键
- word: 单词文本
- phonetic_uk: 英式音标
- phonetic_us: 美式音标
- definition_cn: 中文释义
- definition_en: 英文释义
- level: 难度等级
- frequency: 词频
```

#### 单词本表 (WordBook)
```
- id: 主键
- name: 单词本名称
- description: 描述
- words: 包含的单词列表
- source: 来源（书籍/网页/手动）
- created_at: 创建时间
```

#### 阅读记录表 (ReadingRecord)
```
- id: 主键
- book_name: 书名
- file_path: 文件路径
- progress: 阅读进度
- bookmarks: 书签位置
- last_read_at: 最后阅读时间
```

## 5. 界面设计

### 5.1 主要界面

#### 5.1.1 主界面
- 左侧导航栏：阅读器、单词本、设置
- 中间内容区：根据导航显示不同内容
- 右侧单词详情面板：点击生词时显示

#### 5.1.2 阅读器界面
- 顶部工具栏：打开文件、字体设置、主题切换
- 中间阅读区：文本内容，生词高亮显示
- 底部状态栏：阅读进度、当前页码

#### 5.1.3 单词本界面
- 单词列表：显示收藏的单词
- 筛选排序：按时间、难度、字母排序
- 导出按钮：导出为不同格式

#### 5.1.4 设置界面
- 词汇水平设置
- 界面外观设置
- 数据管理（备份/恢复）

### 5.2 交互设计
- 点击生词：右侧显示单词详情
- 双击生词：添加到单词本
- 右键菜单：复制、发音、添加到单词本
- 快捷键：翻页、收藏、查词
